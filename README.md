# 🏆 DevQuiz: C & C++ Mastery

DevQuiz is a sleek, responsive, and interactive browser-based evaluation engine designed to test proficiency in C and C++ programming. Engineered with a dark-themed, utility-first user interface, it provides developers with real-time feedback, randomized question tracking, and advanced post-evaluation metrics.

## ⚡ Features

- **Multi-Format Testing Framework:** Supports three distinct evaluation modes:
  - **Single Select:** Standard multiple-choice logic.
  - **Multi-Select:** Validates precise set-matching for complex architecture questions.
  - **Exact Match Parsing (Fill-in-the-blank):** Tests syntactic accuracy by parsing exact string inputs.
- **Dynamic Session Configuration:** Filter evaluations by language target (`C` or `C++`) and difficulty depth (`Basic` or `Intermediate`).
- **Live Telemetry & Tracking:** Features a synchronized global execution timer and an animated progression tracking bar.
- **Advanced Performance Analytics:** Uses **Chart.js** to map and visualize post-quiz metrics:
  - **Radar Chart:** Maps your accuracy footprint across granular sub-topics (e.g., Pointers, Smart Pointers, Memory Mgmt).
  - **Bar Chart:** Breaks down chronological time expenditure per sub-topic.
- **Persistent Leaderboard:** Natively caches and ranks global session scores and completion times using browser `localStorage`.

## 🛠️ Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** Tailwind CSS (Utility-first framework via CDN)
- **Data Visualization:** Chart.js (Dynamic canvas rendering via CDN)
- **Core Logic:** Vanilla JavaScript (ES6+ Architecture, state-driven UI updates)
